# Banker's Algorithm (OS Project)

This project implements the Banker's Algorithm in C.

## Features
- Deadlock Avoidance
- Safe Sequence Detection

## Input
- Number of processes
- Allocation matrix
- Max matrix
- Available resources

## Output
- Safe sequence
- Safe/Unsafe state

## 🧪 Sample Input


Enter number of processes: 5
Enter number of resources: 3

Allocation:
0 1 0
2 0 0
3 0 2
2 1 1
0 0 2

Max:
7 5 3
3 2 2
9 0 2
2 2 2
4 3 3

Available:
3 3 2


---

## ✅ Sample Output


System is in SAFE state
Safe sequence: P1 P3 P4 P0 P2
